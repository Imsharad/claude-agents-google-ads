# Google Ads Automation Agent

This project is a Google Ads automation agent.

## Setup

This project requires Python 3.9 or higher.

1.  **Create and activate a virtual environment:**

    It is recommended to use a virtual environment. You can use `venv` or `uv`.

    Using `uv`:
    ```bash
    uv venv
    source .venv/bin/activate
    ```

    Using `venv`:
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

2.  **Install the dependencies:**

    With the virtual environment activated, install the required packages using `uv` or `pip`:

    Using `uv`:
    ```bash
    uv pip install -r requirements.txt
    ```

    Using `pip`:
    ```bash
    pip install -r requirements.txt
    ```
